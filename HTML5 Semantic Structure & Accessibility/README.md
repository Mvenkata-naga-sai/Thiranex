🌐 Personal Portfolio Website (Accessible & Semantic HTML5)
A fully accessible, multi-page personal portfolio website built using modern semantic HTML5 and designed to meet WCAG accessibility standards and achieve 100/100 Lighthouse scores for Accessibility and SEO.

📌 Project Overview
This project demonstrates best practices in:

Semantic HTML5 structure
Web accessibility (WCAG compliant)
SEO optimization
Clean, maintainable code structure

The website includes multiple pages such as:
- Home (index.html)
- About (about.html)
- Projects (projects.html)
- Contact (contact.html)
 
✨ Key Features:
   Semantic HTML5 Structure

Uses proper semantic elements:
- <header>
- <nav>
- <main>
- <section>
- <article>
- <footer>

Accessibility (WCAG Compliant):
- ARIA roles and labels implemented
- Screen reader-friendly structure
- Proper heading hierarchy (h1 → h6)
- Alt text for all images
- Keyboard (Tab) navigable UI
- Accessible forms with labels and error hints

SEO Optimization:
- Meta tags for description and keywords
- Proper document title structure
- Structured content hierarchy
- Clean URL structure
- Mobile-friendly viewport settings

Accessible Contact Form:
- Fully keyboard navigable
- Proper <label> associations
- Required field indicators
- ARIA attributes for validation
- Focus management for usability

🛠️Technologies Used:
- HTML5 (Semantic)
- CSS3 (Responsive design)
- No frameworks (pure fundamentals)
