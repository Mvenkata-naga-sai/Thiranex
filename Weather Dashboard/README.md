# 🌦️ Task Submission: Real-Time Weather Dashboard

## 📌 Task Title

Build a real-time Weather Dashboard by fetching and processing JSON data from a public REST API using asynchronous JavaScript.

---

## 🎯 Objective

The objective of this project is to develop a dynamic weather dashboard that retrieves real-time weather data from a public API and displays it interactively using modern JavaScript techniques like `fetch` and `async/await`.

---

## 🚀 Key Features Implemented

* ✅ **Real-Time Data Fetching**

  * Used Fetch API with `async/await`
  * Retrieves live weather data from a REST API

* ✅ **Error Handling**

  * Handles network failures gracefully
  * Displays user-friendly error messages

* ✅ **JSON Data Processing**

  * Parses complex nested JSON responses
  * Extracts key weather metrics efficiently

* ✅ **Dynamic UI Rendering**

  * Updates weather data without page reload
  * Displays results instantly based on user input

* ✅ **Search Functionality**

  * Users can search weather by city name
  * Supports real-time input-based queries

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the dashboard
* **CSS3** – Styling and layout
* **JavaScript (ES6+)** – Logic, API calls, DOM manipulation
* **REST API** – Weather data source

---

## 📱 Responsiveness

The application is designed to work across:

* 📱 Mobile devices
* 📲 Tablets
* 💻 Desktop screens

---

## ⚙️ Application Workflow

1. User enters a city name
2. Application sends a request to the weather API
3. Receives JSON response
4. Parses required data such as:

   * Temperature
   * Humidity
   * Wind Speed
5. Dynamically updates the UI with fetched data
6. Displays error message if city is invalid or network fails

---

## 📌 Expected Outcome

A fully functional weather dashboard that:

* Provides accurate, real-time weather information
* Responds instantly to user input
* Handles errors efficiently
* Enhances user experience with dynamic updates
